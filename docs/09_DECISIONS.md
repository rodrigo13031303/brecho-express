# Decisões de Arquitetura e Produto - Brechó Express

## Decisões de arquitetura
- Flutter para app mobile.
- Riverpod para estado e injeção de dependência.
- GoRouter para navegação declarativa.
- Dio para comunicação HTTP.
- Oracle + ORDS para backend e APIs REST.
- Arquitetura DDD com Feature First e Linguagem Ubíqua.

## Decisões de produto
- O mesmo app atende cliente e brechó.
- Plataforma nacional de economia circular.
- Marketplace híbrido com Brechó Gratuito e Brechó Plus.
- Checkout baseado em Purchase Request e pagamento após confirmação.
- Pedido pode gerar vários shipments.
- Avaliação e reputação focadas no brechó.
- Suporte a doações, desapego e eventos temporários.

## Decisões de design
- Tema Material 3 com azul claro como cor principal.
- Tokens centralizados para consistência.
- Design System preservado e aplicado em toda a interface.
