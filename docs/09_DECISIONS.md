# Decisões de Arquitetura e Produto - Brechó Express

## Decisões iniciais
- Flutter para app mobile.
- Riverpod para estado e injeção de dependência.
- GoRouter para navegação declarativa.
- Dio para comunicação HTTP.
- Oracle + ORDS para backend e dados.
- Clean Architecture simplificada para separar camadas.

## Decisões de produto
- O mesmo app atende cliente e brechó.
- MVP focado em Curitiba.
- Entrega rápida como proposta de valor.
- Sustentabilidade como elemento de comunicação.

## Design
- Tema Material 3 com azul claro como cor principal.
- Tokens centralizados para consistência.
