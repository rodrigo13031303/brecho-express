# Prompts do Projeto Brechó Express

## Objetivo
Manter um conjunto organizado de prompts para tarefas de IA e geração de documentação, alinhado à visão nacional, arquitetura DDD e linguagem ubíqua do projeto.

## Prompts Sugeridos
- Prompt para gerar telas de login e cadastro de conta/perfil.
- Prompt para descrever o Design System e experiência do marketplace.
- Prompt para criar documentação técnica alinhada ao domínio.
- Prompt para elaborar API, contratos de backend e fluxos de Purchase Request.
- Prompt para criar conteúdo sobre economia circular, Brechó Plus e entrega inteligente.

## Uso
- Use este arquivo como fonte inicial para criar solicitações de IA.
- Atualize os prompts ao evoluir o produto.
- Priorize a terminologia oficial do Brechó Express.
