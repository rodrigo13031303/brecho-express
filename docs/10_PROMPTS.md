# Prompts do Projeto Brechó Express

## Objetivo
Manter um conjunto organizado de prompts para tarefas de IA e geração de documentação.

## Prompts Sugeridos
- Prompt para gerar telas de login e home.
- Prompt para descrever o Design System.
- Prompt para criar documentação técnica.
- Prompt para elaborar API e contratos de backend.

## Uso
- Use este arquivo como fonte inicial para criar solicitações de IA.
- Atualize os prompts ao evoluir o produto.
