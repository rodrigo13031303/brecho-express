# Contexto de IA para o Brechó Express

## Objetivo
Fornecer contexto para assistentes de IA auxiliarem no desenvolvimento e documentação do projeto.

## Sobre o Produto
Brechó Express é um marketplace mobile de brechós, com foco inicial em Curitiba. O aplicativo atende dois perfis: Cliente e Brechó.

## Tecnologias
- Flutter
- Riverpod
- GoRouter
- Dio
- Oracle
- ORDS

## Arquitetura
- Clean Architecture simplificada
- Organização por feature
- Tema e tokens centralizados

## Uso de IA
- Gerar documentação de produto
- Sugerir design e usabilidade
- Ajudar com casos de teste e arquitetura
- Criar conteúdos de roadmap e requisitos
